└─ README.md

1. **src/app.py 작성 및 첫 commit**
   - commit 메시지: `ADD: app.py`
   - 기능: **a 변수에 문자값을 할당함.**

2. **두 번째 commit**
   - commit 메시지: `MODIFY: b`
   - 수정/추가한 내용: **b 변수를 추가함**
3. **세 번째 commit**
   - commit 메시지: `FIX: b to c`
   - 수정/추가한 내용: **b 변수를 c로 수정함.**
